1. How the AI Extraction Works

The flow when someone uploads a COA

1
User picks a file (JPG, PNG, or PDF) in the upload modal on your site
2
The file is converted to base64 and sent to your Supabase Edge Function called extract-coa
3
The edge function sends the image to Anthropic Claude Haiku with instructions to read the COA document
4
Claude reads the document and returns fields: batch number, lab name, purity, date, etc.
5
Those fields auto-fill the upload form. User reviews and submits.

⚠ Your API key lives in Supabase — NOT in the website code

The Anthropic API key is stored as a secret inside your Supabase Edge Function. It is never visible in your website's HTML or JavaScript. This is correct and safe.

The coa-extraction-tests.html file is a dev tool — never put your key in that file and never upload it to GitHub.

3. Managing Your API Key

🔴 If your key gets exposed (e.g. pushed to GitHub)

  1. Go to console.anthropic.com/settings/keys
  2. Click Delete on the exposed key immediately
  3. Click + to create a new key — copy it (shown only once)
  4. Go to Supabase → Edge Functions → extract-coa → Secrets
  5. Update ANTHROPIC_API_KEY with the new key → Save
  6. Click Deploy on the edge function

How to rotate key (routine maintenance)

Same steps as above. Create new key first, update Supabase, then delete old key. Never delete first or the function breaks.

4. How to Switch to the FREE Version (No AI)

If costs get too high or you want to remove AI extraction entirely, follow these steps. Users will still be able to upload COAs — they just fill in the fields manually.

Step 1 — Disable the Supabase Edge Function

  1. Go to supabase.com → your project
  2. Left sidebar → Edge Functions
  3. Click extract-coa
  4. Replace the entire code with this no-op function:
import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'

const CORS = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
  'Access-Control-Allow-Methods': 'POST, OPTIONS',
}

serve(async (req) => {
  if (req.method === 'OPTIONS') {
    return new Response('ok', { headers: CORS })
  }
  return new Response(
    JSON.stringify({ error: 'AI extraction disabled — please fill fields manually.' }),
    { status: 503, headers: { ...CORS, 'Content-Type': 'application/json' } }
  )
})

Click Deploy. AI extraction is now off. The upload form still works — users just type the fields.

Step 2 — Optional: Hide the AI panel in the website

Open index.html, find this line and add the style shown:

// Find this function and add one line at the top:
async function triggerAIExtraction() {
  return; // <-- ADD THIS LINE to disable AI extraction calls
  ...

This stops the site from even trying to call the edge function, so there are zero API calls.

5. How to Switch BACK to Paid AI Version

  1. Go to console.anthropic.com/settings/keys → create a new key
  2. Go to Supabase → Edge Functions → extract-coa → Secrets
  3. Add secret: Name = ANTHROPIC_API_KEY · Value = your new key → Save
  4. Replace the edge function code with the full Claude version (see Section 6 below)
  5. Click Deploy
  6. If you added return; to index.html, remove that line
  7. Test by uploading a COA — fields should auto-fill

6. Full Supabase Edge Function Code (Claude / Paid)

Keep this as your reference. This is the complete working code for the extract-coa edge function.

import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'

const CORS = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
  'Access-Control-Allow-Methods': 'POST, OPTIONS',
}

const EXTRACTION_PROMPT = `You are a precise document parser for Certificates of Analysis (COA)
in the pharmaceutical/research peptide industry.

Extract ONLY what is explicitly visible in the document.
Do NOT infer, guess, or fill in missing information.

Return a JSON object with exactly these fields:
{
  "vendor_name":   { "value": "", "confidence": 0.0, "source_text": "" },
  "product_name":  { "value": "", "confidence": 0.0, "source_text": "" },
  "batch_number":  { "value": "", "confidence": 0.0, "source_text": "" },
  "testing_date":  { "value": "", "confidence": 0.0, "source_text": "" },
  "lab_name":      { "value": "", "confidence": 0.0, "source_text": "" },
  "lab_contact":   { "value": "", "confidence": 0.0, "source_text": "" },
  "sample_id":     { "value": "", "confidence": 0.0, "source_text": "" },
  "compound_name": { "value": "", "confidence": 0.0, "source_text": "" },
  "purity_result": { "value": "", "confidence": 0.0, "source_text": "" },
  "test_type":     { "value": "", "confidence": 0.0, "source_text": "" }
}

Rules:
- confidence: 0.0=not found, 0.5=unclear, 0.9+=clearly visible
- source_text: copy exact text from the document
- Return ONLY the JSON object — no markdown, no explanation`

serve(async (req) => {
  if (req.method === 'OPTIONS') {
    return new Response('ok', { headers: CORS })
  }
  try {
    const anthropicKey = Deno.env.get('ANTHROPIC_API_KEY')
    if (!anthropicKey) {
      return new Response(JSON.stringify({ error: 'ANTHROPIC_API_KEY secret not set' }), {
        status: 500, headers: { ...CORS, 'Content-Type': 'application/json' }
      })
    }
    const { fileData, mediaType } = await req.json()
    if (!fileData || !mediaType) {
      return new Response(JSON.stringify({ error: 'Missing fileData or mediaType' }), {
        status: 400, headers: { ...CORS, 'Content-Type': 'application/json' }
      })
    }
    const safeType = ['image/jpeg','image/png','image/gif','image/webp','application/pdf']
      .includes(mediaType) ? mediaType : 'image/jpeg'
    const sourceBlock = mediaType === 'application/pdf'
      ? { type: 'document', source: { type: 'base64', media_type: 'application/pdf', data: fileData } }
      : { type: 'image', source: { type: 'base64', media_type: safeType, data: fileData } }
    const claudeResp = await fetch('https://api.anthropic.com/v1/messages', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': anthropicKey,
        'anthropic-version': '2023-06-01'
      },
      body: JSON.stringify({
        model: 'claude-haiku-4-5',
        max_tokens: 1024,
        system: EXTRACTION_PROMPT,
        messages: [{
          role: 'user',
          content: [
            sourceBlock,
            { type: 'text', text: 'Extract COA fields. Return only the JSON object.' }
          ]
        }]
      })
    })
    if (!claudeResp.ok) {
      const errText = await claudeResp.text()
      throw new Error('Anthropic API error ' + claudeResp.status + ': ' + errText.substring(0,300))
    }
    const claudeData = await claudeResp.json()
    const rawText = claudeData.content[0].text
    const match = rawText.match(/\{[\s\S]*\}/)
    if (!match) throw new Error('No JSON in response: ' + rawText.substring(0,200))
    return new Response(JSON.stringify(JSON.parse(match[0])), {
      headers: { ...CORS, 'Content-Type': 'application/json' }
    })
  } catch (e) {
    console.error('extract-coa error:', e)
    return new Response(JSON.stringify({ error: e.message }), {
      status: 500, headers: { ...CORS, 'Content-Type': 'application/json' }
    })
  }
})
